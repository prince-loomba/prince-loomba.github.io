# Crime SQL Lab

A free, static SQL mystery game that lets players solve escalating crime cases directly in the browser.

## Features
- Four cases from easy to expert difficulty
- Modern dark UI with a polished navigation experience
- No sign-in, no paid services, and no backend required
- Ready for GitHub Pages deployment

## Run locally
1. Open the repository folder in a browser, or serve it with a simple static server.
2. For example: `python -m http.server 8000`
3. Visit http://localhost:8000

## GitHub Pages
- Push the repository to GitHub.
- Enable GitHub Pages from the repository settings.
- Choose the root folder or the docs folder as the deployment source.
- The app works as a static site without a database or server.
